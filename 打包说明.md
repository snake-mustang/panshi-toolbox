# 打包给用户使用（用户无需安装）

用 PyInstaller、Nuitka 等把项目打成 exe 时，**在打包环境里先装好所有依赖**，再执行打包命令。这样依赖会一起打进 exe 或 `_internal` 目录，**用户拿到 exe 后直接运行即可，不用再装 Python 或任何插件**。

- **用户只需安装/运行本应用**：打好的 exe 自带 OCR，无需再装任何东西。
- **开发**：任意 Python（含 3.13）执行 `pip install -r requirements.txt` 即可运行，OCR 未装时会提示。
- **打包带 OCR**：请用 **Python 3.11 或 3.12**，执行 `pip install -r requirements.txt` 和 `pip install -r requirements-ocr.txt` 后打包。

## 1. 打包前在「打包机」上装依赖

```bash
# 打包带 OCR 请用 Python 3.11 或 3.12
cd app-toolbox
python -m pip install -r requirements.txt
python -m pip install -r requirements-ocr.txt
python run_webview.py
# 关掉窗口后执行打包
```

## 2. 用 PyInstaller 打包示例

```bash
# 安装 PyInstaller
python -m pip install pyinstaller

# 单文件 exe（体积大、启动略慢）
pyinstaller --onefile --windowed --name "IT工具箱" run_webview.py --add-data "static;static"

# 或 目录模式（启动快，推荐）
pyinstaller --windowed --name "IT工具箱" run_webview.py --add-data "static;static"
```

打包完成后，exe 在 `dist/IT工具箱/` 下，**把整个文件夹发给用户即可，用户无需安装任何东西**。

## 3. 要点

- **用户不用安装**：依赖和 OCR（RapidOCR）都会打进 exe，用户只运行我们的应用即可使用全部功能。
- **打包机装一次即可**：用 3.11 或 3.12 执行上述两条 pip 后打包即可。
